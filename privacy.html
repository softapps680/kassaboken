Integritetspolicy för Kassaboken

Senast uppdaterad: 20231030

Introduktion

Välkommen till Kassaboken. Denna integritetspolicy är utformad för att hjälpa dig förstå hur vi och vår reklampartner AdMob samlar, använder, delar och skyddar dina data när du använder vår mobila applikation.

Information som samlas in

Vi kräver inte att användare skapar konton eller lämnar personlig information för att använda vår app. Däremot kan vår reklampartner, AdMob, samla följande information för reklam och analytiska ändamål, såsom:

Enhetsinformation: AdMob kan samla information om den enhet du använder för att komma åt vår app, såsom enhetstyp, operativsystem och unika enhetsidentifierare.

Användningsdata: AdMob kan samla information om hur du använder vår app, inklusive vilka funktioner du interagerar med och dina interaktioner inom appen.

AdMob kan använda den insamlade informationen för följande ändamål: Målriktad reklam: AdMob kan använda denna information för att visa personligt anpassade annonser baserat på dina intressen och webbaktivitet. Analys: AdMob kan använda data för att förstå användarbeteende och optimera leverans av annonser i vår app.

AdMobs integritetspolicy AdMob är en tredjepartstjänst och deras datainsamling och användning regleras av deras egen integritetspolicy. Du kan granska AdMobs integritetspolicy här..

Dataskydd

Vi tar dataskydd på allvar och implementerar rimliga säkerhetsåtgärder för att skydda dina uppgifter. Dock, var medveten om att ingen metod för dataöverföring över internet eller elektronisk lagring är helt säker, och vi kan inte garantera absolut säkerhet för dina uppgifter.

Ändringar i denna integritetspolicy

Vi kan uppdatera vår integritetspolicy med jämna mellanrum. Vi kommer att meddela dig om eventuella ändringar genom att publicera den nya integritetspolicyn på denna sida. Det är ditt ansvar att regelbundet granska denna integritetspolicy.

Kontakta oss

Om du har frågor eller funderingar angående denna integritetspolicy eller vår datapraxis, vänligen kontakta oss på softapps987@gmail.com.

Laglig efterlevnad

Vi följer alla tillämpliga integritetslagar och föreskrifter.

Genom att använda vår app godkänner du villkoren i denna integritetspolicy. Om du inte är överens med policyn, vänligen avstå från att använda vår app.
